# Retail Sales EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to identify sales trends, customer segments, and product performance.

## Objectives
- Understand the structure of the dataset
- Perform data cleaning and preprocessing
- Analyze monthly and quarterly sales trends
- Explore customer segment distribution
- Identify top-selling products
- Visualize key business insights

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Findings
- Analyzed sales trends over time.
- Compared customer segments.
- Identified top-performing products.
- Created visualizations to support business decision-making.

## Files
- `Retail_Sales_EDA.ipynb` – Jupyter Notebook containing the analysis.
- `Sample_Superstore.csv` – Dataset used for the project.
